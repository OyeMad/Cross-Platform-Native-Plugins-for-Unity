Detailed Description

Provides interface to communicate with game server about local user's progress towards completing achievement.

By default, when an achievement is completed, a notification banner is displayed to the user. If your game wants to display its own iterface, you can prevent it by setting Show Default Achievement Completion Banner property in Game Services Settings to NO.

Note
Your game must authenticate the local user before using any features.
