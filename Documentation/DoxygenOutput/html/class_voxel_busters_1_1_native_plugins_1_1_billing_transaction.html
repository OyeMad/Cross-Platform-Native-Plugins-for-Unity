Detailed Description

Provides transaction information of the purchased product.

Member Function Documentation

void UpdateVerificationState ( eBillingTransactionVerificationState  _newState)

Call this method to manually update the state, after custom receipt verfication is completed.

Parameters
_newStateThe new state for this transaction.

Note that, calling this method will cause product purchase inventory to be updated and followed by Billing.DidReceiveTransactionInfoEvent is raised.
