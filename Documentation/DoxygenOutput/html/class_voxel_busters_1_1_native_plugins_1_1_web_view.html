Detailed Description

Provides interface to embed web content in your application.

Package includes WebView prefab, which is placed under "Assets/VoxelBusters/NativePlugins/Prefab" folder. Drag and drop WebView prefab into heirarchy and send requests to display local content or content that is loaded from the network.
