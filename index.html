Folder Structure

All Voxel Busters plugins will be under VoxelBusters folder and Common folder will be same for all future plugins too.

Android

In Plugins/Android/native-plugins_lib/AndroidManfest.xml file, replace "PASTE_BUILD_IDENTIFIER_HERE" with your build identifier.

Copy required entries or rename Assets/Plugins/Android/AndroidManifest_root_activity_settings_sample.txt to AndroidManifest.xml if you don't have already root manifest file. Please check comments in this file for required attributes in your root manifest file.
